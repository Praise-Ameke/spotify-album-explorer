# Spotify Album Explorer

A React and TypeScript web application for discovering artists and exploring albums using the Spotify Web API.

## Current Features

- Reusable `Header` and `SearchBar` components
- Controlled artist search input
- React state management with `useState`
- Parent-to-child communication using props
- Search event handling
- Separate Spotify service layer

## Tech Stack

- React
- TypeScript
- Vite
- CSS
- Spotify Web API

## Project Status

🚧 Currently in development.

The next features will include Spotify API integration, loading states, error handling, album results, and recent search history.
